## Installing

Clone the repo

```
git clone https://github.com/RafaelWeiss/mobile-test.git
```

Change into the up-movies-app project directory

```
cd mobile-test
```

Install project dependencies

```
yarn install
```

And start the application using Expo

```
yarn start
```

Run tests with coverage status

```
yarn test --coverage
```
